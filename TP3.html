1.1- ¿Qué significa CSS?

CSS significa Hojas de Estilo en Cascada, en inglés es Cascading Style Sheets y te dará los estilos (diseño) necesarios para que un sitio web o una aplicación web se vean visualmente bien.


1.2- ¿Para qué sirve el CSS? ¿Qué programa se utiliza para interpretar el CSS?

CSS sirve para generar unb ascpecto bonito y agradable para el espectador/usuario. Algunas de las cosas que podes hacer es:

  1. Definir aspectos visuales de un sitio
  2. Crear características general a todo el sitio o aplición web
  3. Actualización de los aspectos visuales de forma oganizada
  4. Dar colores a los fondos y los textos
  5. Cambiar las tipografías (fuentes) y tamaños a los textos
  6. Aplicar márgenes
  7. Aplicar bordes
  8. Hacer adaptativo un sitio o aplicación web para que se vea de forma correcta en cualquier dispotivo

1.3- ¿A qué se llama regla de CSS?

Se le llama regla de css a los conjuntos selectores, llaves, propiedad y valores.

1.4- ¿Cuál es la anatomía de la regla de CSS? Agregar imagen explicativa y utilizando una lista descriptiva explicar sus elementos.

La anatomia de la regla de css es un conjunto de propiedades y valores que se aplicarán a una o varias etiquetas según necesitemos.

  1. Selector: selecciona el(los) elemento(s) a dar estilo (en este caso, los elementos "p" ). Para dar estilo a un elemento diferente, solo cambia el selector.
  2. Declaración: aplica las propiedades que querés dar al elemento seleccionado en el Selector. En este caso hay una sola regla, pero puede haber más.
  3. Propiedades: es lo que te permite dar un estilo a un elemento HTML. En el ejemplo la propiedad es color que le dará color a las letras de los párrafos. Siempre las propiedades se ubicarán a la izquierda.
  4. Valor de la propiedad: las propiedades deben tener un valor. Cada propiedad tiene valores definidos. Cada valor se ubicará a la derecha.

1.7- Explicar brevemente utilizando una lista ordenada: ¿cómo funciona el CSS?

      Seleccion de elementos
    • CSS utiliza selectores para identificar los elementos HTML a los que se les aplicarán los estilos. Los selectores pueden seleccionar elementos por etiqueta, clase, ID, atributo y otros criterios.

      Declaracion de estilos
    • Una vez seleccionados los elementos, se especifican las propiedades de estilo que se desean aplicar. Por ejemplo, se pueden definir colores, tamaños de fuente, márgenes, bordes, fondos, etc.

      Asociación de estilos
    • Para asociar los estilos con los elementos HTML, se utiliza una regla de estilo. Una regla de estilo consta de un selector y un bloque de declaraciones de estilo. El selector selecciona los elementos a los que se aplicarán los estilos, y el bloque de declaraciones contiene las propiedades y valores de estilo.

      Cascada y especificidad
    • CSS utiliza el concepto de cascada para determinar qué estilo se aplicará a un elemento en caso de que existan múltiples reglas que coincidan con él. La cascada asigna prioridad a las reglas según su especificidad y su orden de aparición en el código.

        • Herencia
      • CSS permite que los estilos se hereden de elementos padre a elementos hijos. Esto significa que los estilos aplicados a un elemento se aplicarán también a sus elementos hijos, a menos que se especifique lo contrario.

        Estilos externos e internos
    • CSS se puede incluir en una página web de diferentes formas. Puede estar dentro de la etiqueta "style" en el propio documento html (estilos internos), o se puede enlazar a un archivo CSS externo mediante la etiqueta "link" (estilos externos). Los estilos externos son útiles cuando se desea aplicar los mismos estilos a varias páginas web.
